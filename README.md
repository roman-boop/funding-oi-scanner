
# 📊 Funding & OI Scanner для BingX

[![Python](https://img.shields.io/badge/Python-3.8%2B-blue?logo=python)](https://www.python.org/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![BingX](https://img.shields.io/badge/Exchange-BingX-ffba00?logo=binance)](https://bingx.com/invite/FZGEZV)

Сканер для фьючерсного рынка **BingX**, который в реальном времени отслеживает **Открытый Интерес (Open Interest)** и **Ставки Финансирования (Funding Rate)**. Помогает находить аномалии, «перегретые» монеты и моменты накопления позиций для принятия торговых решений.

---

## 🚀 Возможности

*   📈 **Ставки финансирования** — вывод текущих ставок по всем доступным парам или по заданному списку.
*   🔍 **Открытый интерес (OI)** — отслеживание текущего OI и его относительного изменения за выбранный период.
*   ⚡ **Фильтрация аномалий** — быстрый поиск монет с экстремально высоким/низким Funding Rate или резким скачком OI.
*   📋 **Сортировка и вывод** — удобная таблица в консоли с основными метриками (цена, объем, OI, ставка).

## 🛠 Установка и запуск

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/roman-boop/funding-oi-scanner.git
    cd funding-oi-scanner
    ```

2.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
    *(Если файла `requirements.txt` ещё нет, установите нужные библиотеки вручную, например: `pip install requests pandas`)*

3.  **Запустите сканер:**
    ```bash
    python bingx_scanner.py
    ```
    *   `bingx_client.py` — модуль для прямого взаимодействия с API BingX.
    *   `bingx_scanner.py` — основной скрипт для запуска сканирования и вывода результатов.

---

## ⚙️ Как это работает

Скрипт подключается к публичным API-эндпоинтам BingX и получает данные по фьючерсным парам (USDT-M, Coin-M). На основе полученной информации рассчитывается:

*   **Текущая ставка финансирования** (Funding Rate) — показывает, в какую сторону платят «лонгисты» или «шортисты».
*   **Изменение OI (Delta OI %)** — рост открытого интереса часто указывает на приток нового капитала в актив, падение — на закрытие позиций или ликвидации.
*   **Соотношение Long/Short** (если доступно в API) — для оценки настроения толпы.

Результаты выводятся в консоль в виде отсортированной таблицы, что позволяет быстро оценить рыночную картину.
